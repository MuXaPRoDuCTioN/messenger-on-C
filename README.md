# Мессенджер на языке C

Курсовой проект по дисциплине «Операционные системы».  
Студент: Киселёв Михаил Олегович, группа ПИ-31.

## Возможности

- Личные сообщения между пользователями
- Групповые чаты
- Цитирование (reply) и пересылка (forward) сообщений
- История переписки (SQLite на сервере, кэш на клиенте)
- Текстовый интерфейс на ncursesw (UTF-8)
- Поддержка русского языка
- Многопоточный сервер (pthread)
- Работа в локальной сети и через интернет

## Зависимости (Ubuntu/WSL)

```bash
sudo apt install build-essential libncursesw5-dev libsqlite3-dev

## Сборка

```bash
make            # собрать сервер и клиент
make clean      # удалить объектные файлы и бинарники
```

## Запуск

```bash
# Сервер (порт по умолчанию 12345)
./server_app

# Клиент
./client_app 127.0.0.1   # или IP-адрес сервера в локальной сети
# После подключения в консоли будет запрошен логин и пароль.
# При первом входе пользователь регистрируется автоматически.
```

## Структура проекта

```
messenger/
├── common/
│   └── protocol.h       — общие константы и типы сообщений
├── server/
│   ├── server.h         — общие типы сервера
│   ├── main.c           — точка входа сервера
│   ├── network.c        — список онлайн-клиентов, рассылка
│   └── db.c             — работа с SQLite (серверная БД)
├── client/
│   ├── client.h         — общие типы клиента
│   ├── main.c           — точка входа клиента
│   ├── network.c        — подключение к серверу, сетевой поток
│   ├── ui.c             — интерфейс на ncursesw
│   ├── local_db.c       — локальный кэш сообщений (SQLite)
│   └── local_db.h       — заголовок локальной БД
└── Makefile
```

## Технологии

| Компонент       | Технология                       |
|-----------------|----------------------------------|
| Язык            | C11                              |
| Сеть            | POSIX сокеты, TCP                |
| Многопоточность | pthreads                         |
| База данных     | SQLite 3 (сервер + локальный кэш)|
| Интерфейс       | ncursesw (широкие символы)       |
| Протокол        | Текстовый, разделитель `\|`      |
