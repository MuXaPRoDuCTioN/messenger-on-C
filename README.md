# Мессенджер на языке C

Курсовой проект по дисциплине «Операционные системы».  
Студент: Киселёв Михаил Олегович, группа ПИ-31.

## Возможности

- Личные сообщения между пользователями
- Групповые чаты
- Цитирование и пересылка сообщений
- История переписки (SQLite)
- Текстовый интерфейс (ncurses)
- Работа в локальной сети и через интернет

## Сборка

```bash
make            # собрать сервер и клиент
make clean      # удалить объектные файлы и бинарники
```

## Запуск

```bash
# Сервер (по умолчанию порт 8080)
./server_app [port]

# Клиент
./client_app [host]
# Пример: ./client_app 127.0.0.1
```

## Структура проекта

```
messenger/
├── common/
│   └── protocol.h       — общие константы и типы сообщений
├── server/
│   ├── main.c           — точка входа сервера
│   ├── network.c/h      — TCP-сокеты, потоки клиентов
│   ├── db.c/h           — работа с SQLite
│   └── server.h         — общие типы сервера
├── client/
│   ├── main.c           — точка входа клиента
│   ├── network.c/h      — подключение к серверу
│   ├── ui.c/h           — интерфейс ncurses
│   └── client.h         — состояние приложения
└── Makefile
```

## Технологии

| Компонент     | Технология                       |
|---------------|----------------------------------|
| Язык          | C11                              |
| Сеть          | POSIX сокеты, TCP                |
| Многопоточность | pthreads                       |
| База данных   | SQLite 3                         |
| Интерфейс     | ncurses                          |
| Протокол      | Текстовый, разделитель `\|`      |
